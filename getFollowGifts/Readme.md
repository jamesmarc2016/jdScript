    |   |-- jd_getFollowGift.py       # 主代码 （必要）

    1.Python3.6+ 环境
    2.兼容ios设备软件：Pythonista 3、Pyto(已测试正常跑，其他软件自行测试)   
    3.Windows exe

    pip3 install requests

    python3 jd_getFollowGift.py
